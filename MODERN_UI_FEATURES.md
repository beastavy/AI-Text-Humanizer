## 📱 **How to Use:**

### **🚀 Launch the Modern App:**
1. **Double-click** `START_MODERN.bat` OR
2. **Run**: `streamlit run main_modern.py`

### **🌐 Access URLs:**
- **Modern UI**: `http://localhost:8501` (or next available port)
- **Original UI**: `http://localhost:8505`

### **📝 Using the App:**
1. **Enter text** in the input area
2. **Choose options** in the sidebar:
   - ✅ **Passive Voice** for formal tone
   - ✅ **Synonym Replacement** for sophisticated words
   - ✅ **Preserve Structure** for formatting
3. **Click "Transform"** and watch the progress
4. **Copy text** using the copy button (no more vanishing!)
