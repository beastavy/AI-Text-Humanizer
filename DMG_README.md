## 📱 How to Use

### Quick Start
1. **Launch** the app from Applications
2. **Paste** or type your AI-generated text in the input area
3. **Choose** your transformation settings
4. **Click** "Humanize" to transform
5. **Copy** or save your humanized text

### Advanced Usage

#### Transformation Modes
- **Light**: Subtle improvements, maintains original style
- **Medium**: Balanced transformation, professional output
- **Heavy**: Maximum humanization, complete rewrite

#### Settings Options
- ✅ **Expand Contractions** - Converts "don't" to "do not"
- ✅ **Add Transitions** - Inserts academic transitions
- ✅ **Enhance Vocabulary** - Uses sophisticated words
- ✅ **Complex Clauses** - Adds subordinate clauses
- ✅ **Passive Voice** - Converts to passive where suitable
- ✅ **Hedging Language** - Adds academic caution

### Keyboard Shortcuts
- `⌘ + O` - Open file
- `⌘ + S` - Save output
- `⌘ + ↵` - Humanize text
- `⌘ + C` - Copy output
- `⌘ + V` - Paste input
- `⌘ + Z` - Undo
- `⌘ + ⇧ + Z` - Redo

## 🆘 Troubleshooting

### App Won't Open
If macOS says the app is from an unidentified developer:
1. Right-click the app in Applications
2. Select "Open" from the menu
3. Click "Open" in the dialog

### Performance Issues
- Close unnecessary applications
- Restart the app
- For large texts, use batch processing

### Text Not Transforming
- Ensure text is in English
- Check transformation settings
- Try different intensity levels

## 💡 Tips & Tricks

### Best Practices
1. **Start with Medium intensity** for balanced results
2. **Review output** before using in important documents
3. **Combine features** for best results
4. **Save presets** for consistent transformations
5. **Use batch mode** for multiple documents

### Pro Tips
- Use Light mode for minor touch-ups
- Enable all features for academic papers
- Disable transitions for creative writing
- Export to RTF to preserve formatting
