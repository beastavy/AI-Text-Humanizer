🚀 3 SIMPLE STEPS TO GET STARTED:
========================================

STEP 1: Install Python (if you don't have it)
-------------------------------------------
1. Go to: https://www.python.org/downloads/
2. Download and install Python 3.10 or newer
3. IMPORTANT: Check "Add Python to PATH" during installation
4. Restart your computer if prompted

STEP 2: Run Setup (install AI components)
------------------------------------------
Double-click this file to install everything:
📁 SETUP.bat

This will automatically:
✅ Create virtual environment
✅ Install all AI libraries (spaCy, NLTK, transformers)
✅ Download language models
✅ Setup advanced features

STEP 3: Start the App
---------------------
Double-click this file to launch:
📁 START.bat

The app will automatically:
✅ Find an available port
✅ Open in your web browser
✅ Load all advanced features
